# Projects

This folder holds mini-projects and your Week 12 capstone — work that's bigger than a single week's lab, or that pulls together material from multiple weeks.

## What Goes Here

- Any mini-project called out in a specific week's instructions
- Your Week 12 capstone: incident report + executive summary, and your final portfolio polish

This folder is empty until a project is assigned — most weeks won't add anything here.

## Structure

Create one subfolder per project, named for what it is:

```
projects/
└── week-12-capstone/
    ├── incident-report.md
    └── executive-summary.md
```

## Format

Same standards as everywhere else in this portfolio: clear technical reasoning, structured thinking, professional tone. This folder is often the first thing a hiring manager looks at closely — treat it that way.
